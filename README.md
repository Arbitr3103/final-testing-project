# Final Testing Project

Это финальный проект по подготовке окружения для автотестирования с использованием PyTest и Selenium.

## Содержимое

- `conftest.py`: Настройки и фикстуры для тестов.
- `test_main_page.py`: Тесты для главной страницы.
- `requirements.txt`: Зависимости проекта.
- `__init__.py`: Файл для распознавания папки как модуля.
